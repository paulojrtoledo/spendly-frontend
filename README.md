# Elo Bank — Frontend

Modern and responsive frontend for Elo Bank, a digital banking platform built for study and portfolio purposes.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- React Router DOM v7
- Axios

## Features (in progress)

- [x] Login page with CPF and password validation
- [ ] Customer registration
- [ ] Dashboard
- [ ] Transactions
- [ ] Pix key management
